## Stack

1. Node
2. Grunt
3. Require
4. Jasmine
5. Protractor
6. Karma
7. Express.js
8. Kraken.js

## Deployment

- Deployed to free Heroku tier and unit tested through Travis CI. 
- Static code analysis and coverage reports by Code Climate.
